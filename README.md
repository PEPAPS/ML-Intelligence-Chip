# ML-Intelligence-Chip
This repository contains work for the Machine Learning Intelligent Chip Design course, which focuses on designing and modeling hardware accelerators for machine learning workloads using SystemC. The course introduces how neural-network computation can be mapped onto a hardware architecture using processing elements, a mesh-based Network-on-Chip, memory hierarchy, and controller-based scheduling. Topics include CNN accelerator design, PE computation, router and packet/flit communication, DRAM-based storage, AXI4 bus behavior, AXI4 DMA transfers, burst read/write transactions, VALID/READY handshaking, SRAM/local buffering, data reuse, and hardware/software trade-off analysis. The final project extends the earlier ROM-based AlexNet accelerator into a DRAM/DMA-based NoC accelerator and compares a baseline design with an optimized design based on PE mapping, memory movement, NoC utilization, MAC resources, and execution-cycle modeling.
